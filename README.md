Stock Portfolio Tracker 📈

A simple Python project that helps users calculate the total value of their stock investments using predefined stock prices.

Features

- Track multiple stocks
- Calculate investment value automatically
- Supports AAPL, TSLA, GOOG, and MSFT
- Saves portfolio value to a text file
- Beginner-friendly project

Technologies Used

- Python 3

How to Run

1. Download or clone the repository.
2. Open the terminal in the project folder.
3. Run the program:

python stock_portfolio_tracker.py

Example Stocks

- AAPL
- TSLA
- GOOG
- MSFT

Output

- Displays individual stock investments
- Calculates total portfolio value
- Saves the result in "portfolio.txt"
